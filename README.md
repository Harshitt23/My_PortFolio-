⚡ Getting Started
Prerequisites

Make sure you have the following installed:

Node.js
 (v14 or higher)

npm or yarn

Installation
git clone https://github.com/Harshitt23/portfolio.git
cd portfolio
npm install

Run Locally
npm run dev


Then open your browser at:
👉 http://localhost:3000

🌍 Deployment

Deployed on Vercel for fast, serverless performance.
To deploy your own version:

npm run build
vercel --prod
